Marriott Portal Invoice Automation

A comprehensive solution for automating invoice downloads from the Marriott vendor portal

Overview

This documentation provides a comprehensive overview of the automated process for downloading invoices from the Marriott portal. The automation is designed to retrieve invoices systematically, providing error handling, notifications, and integration with SharePoint.

Component Details

Master Process

Entry point for automation that parses command-line arguments

Marriott Driver

Core orchestration component that coordinates the entire workflow

Login Process

Handles authentication with OTP verification for secure access

Lookup Data

Processes Excel data files for invoice identification

Invoice Selection

Selects appropriate invoices based on filters and criteria

Invoice Download

Handles PDF file retrieval and validation

1. Master Process (master.py)

Entry point for the automation that parses command-line arguments and initializes the Marriott Driver.

python master.py --module marriott --process invoice_download --email_list recipients@email.com --is_prod True --month 05-2023 --run_flow True

2. Marriott Driver (marriott_driver.py)

Core component that orchestrates the entire automation process:

  • Initializes WebDriver
  • Manages the login process
  • Coordinates data loading and invoice selection
  • Handles downloads and validation

3. Login Process (login_page.py)

  1. Navigate to Marriott login page
  2. Enter username and password from Azure Key Vault
  3. Handle OTP authentication
  4. Verify successful login

4. Lookup Data Processing (load_lookup_data.py)

  1. Download lookup file from SharePoint or use local file
  2. Parse lookup data for properties and AR types
  3. Prepare data for invoice selection

5. Invoice Selection (invoice_selection.py)

  1. Select invoice date (either specified or derived from month parameter)
  2. Select appropriate AR types based on lookup data
  3. Apply filters for targeted invoice retrieval

6. Invoice Download (invoices_download.py)

  1. Construct URL with selected filters
  2. Navigate to invoice download page
  3. Export invoices in PDF format
  4. Validate downloaded files

Error Handling

The system includes comprehensive error handling mechanisms:

Error Type Handling Mechanism
Login Failures Multiple retry attempts with screenshots
OTP Authentication Issues Validation checks with backup locators
Lookup File Processing Errors Exception handling with error notifications
Download Failures Validation and retry mechanisms
SharePoint Transfer Issues Error logging and email notifications

Conclusion

The Marriott Portal Invoice Automation system provides a robust, efficient solution for automatically retrieving invoices from the Marriott vendor portal. This automation significantly reduces manual effort, minimizes human error, and ensures timely processing of invoices.

Key Benefits

  • Elimination of manual login and navigation
  • Standardized processing based on business rules
  • Secure credential management through Azure Key Vault
  • Comprehensive error handling with notifications
  • Seamless integration with SharePoint
  • Detailed logging for audit purposes

Continuous improvements and maintenance ensure that the automation adapts to changes in the Marriott portal and evolving business requirements.
